# Övning

Tänk dig på två filer med följande innehåll:

```txt
12.50
17.50
```

```txt
2
10
```

Bygg en lösning där man kan välja var och en av dessa två filer och generera en ny fil med följande innehåll:

```txt
12.50x2=25.00
12.50x10=125.00
17.50x2=35.00
17.50x10=175.00
```
